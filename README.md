* Fullbright / Decals / Details / Specular / Specular Mode / AA Shadows / Normals
* Laser
* Viewmodel X Y Z Position
* FOV & FOV Scale
* Camera Bob
* Movie config + Export CFG

All settings listed above are saved in ``profile.xml``
